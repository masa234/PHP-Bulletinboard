# Usage

Apacheのドキュメントルートにこのフォルダをコピー
その後、

①
`https://github.com/masa234/php-bulletinboard`

②
xamppを立ちあげ、Apacheを起動。

③
お使いのブラウザで
`http://localhost/php-bulletinboard/index.php`に接続してください。
閲覧できると思います。
